# Snake Royale

## About

Snake Royale is an exciting twist on the classic Snake game. In this game, you'll not only grow your snake but also gain the ability to eat other snakes by consuming a special type of food. Survive in a competitive snake-eat-snake world and watch the simulation unfold before your eyes!

This project was created by Noah Threlfall-Holmes as a Year 1 university project for Computer Games Programming. It is written in C++ using the SFML library.

## How to Play

- Player 1 controls:
  - W: Move Up
  - A: Move Left
  - S: Move Down
  - D: Move Right

- Player 2 controls (Optional):
  - Arrow Keys: Up, Left, Down, Right

If you prefer using arrow keys, you can enable Player 2 in the game options and let Player 1 die when you start.

## Features

- Engaging and competitive gameplay.
- Unique "snake-eating" ability.
- Multiplayer support with Player 1 and Player 2 options.
- Beautiful graphics and smooth animations.

## Installation

To play Snake Royale, follow these simple steps:

1. Clone this repository to your local machine.

2. Build the game using your preferred C++ compiler.

3. Run the executable file.
